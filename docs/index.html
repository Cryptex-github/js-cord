▸ onSlashCommand(command, callback)
js-cord/client/Client.js

Listens for when a slash command is invoked.

Callback can take zero, one (ctx: SlashContext) parameter, or two (ctx: SlashContext, options: Object) parameters.

You must supply a slash command or it's ID in this case. You can query slash commands by it's name via Client#getSlashCommandNamed.

onSlashCommand(command: (SlashCommand | string), callback: function)
Parameters
command ((SlashCommand | string)) The slash command or it's ID to listen for.
callback (function) The callback of the command.
▸ createSlashCommand(command, guilds, callback)
js-cord/client/Client.js

Creates a global slash command. You can also create guild slash commands here, too.

createSlashCommand(command: SlashCommand, guilds: (Guild? | string | Array<(Guild | string)>), callback: function?): SlashCommand
Parameters
command (SlashCommand) The slash command to create.
guilds ((Guild? | string | Array<(Guild | string)>)) The guild or array of guilds that this slash command will be created in. Leave blank for global.
callback (function?) The callback for when this command is invoked.
Returns
SlashCommand: The slash command created.
